🔥 Connect to Firebase (Free)
One-time setup. Data will sync across all your devices.
Instructions — takes 3 minutes:
1. Go to console.firebase.google.com
2. Create a new project → name it "chandrawati-opd"
3. Skip Google Analytics → Create project
4. Click ⚙️ (Settings) → Project Settings
5. Under "Your apps" → click </> (Web) → Register app
6. Copy your firebaseConfig object values below
7. Go to Realtime Database → Create database → Start in test mode
🔑 Enter Firebase Config
Paste the values from your Firebase project settings.
⚠ Find this in Firebase Console → Build → Realtime Database → Data tab. Must start with https:// and end with .firebasedatabase.app
ℹ Database URL ends in .firebasedatabase.app — find it under Realtime Database → Data tab
